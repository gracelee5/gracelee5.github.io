# 속성

        <a href="https://www.naver.com/">네이버</a>

    새로운 페이지에서 열리게 하고 싶으면

        <a href="https://www.naver.com/" target="_blank">네이버</a>﻿

    title="네이버로 연결~~" -> 마우스 포인터 올려놨을 때 내용 표시

    브라우저는 줄바꿈을 무시한다.   
    따라서 목록으로 표현하고 싶으면

        <li>기본 문법</li>
        <li>하이퍼 텍스트와 속성</li>
        <li>리스트와 태그의 중첩</li>

## 문서의 구조
---

* 탭 제목을 표시하려면

       <title>수업 소개</title>     
* 글자 깨질 경우

        <meta charset = "utf-8">

<br/><br/>

모든 태그는 head 태그나 body 태그로 구분이 된다.   

**head** 본문이 아닌 태그
**body** 본문 부분

html 태그는 head, body 태그를 감싸준다.

ex)

    <html>
    <head>
       <title>HTML 수업 소개</title>
       <meta charset = "utf-8">
    </head>
    <body>
    <h1>HTML</h1>
    <ol>
    <li>기술 소개</li>
    <li>기본 문법</li>
    <li>하이퍼 텍스트와 속성</li>
    <li>리스트와 태그의 중첩</li>
    </ol>

    <h2>선행학습</h2>

    파이팅!!

    </body>
    </html>
